# Enterprise Network Automation Lab

This project implements a fully automated, multi-site enterprise network lab.

## Key Principles
- Infrastructure as Code
- Automation-first (no manual CLI configuration)
- Git as the single source of truth
- CI/CD-driven deployment and validation

## Scope
- 4 sites (HQ + 3 branches)
- Multi-protocol routing
- VLANs and VRFs
- Multi-vendor simulation
- Designed to run on a single modest PC
